## 📝 Project Overview
**DeshCare Vaccine** is a web application designed to streamline vaccination management.  
The system allows users to **register, schedule, and track vaccinations**, while enabling **admins and operators** to manage patient records efficiently.

---

## 🚀 Features
- **👥 Role-Based Dashboards:** Separate access for admins, operators, and patients.  
- **📅 Schedule Management:** Book and track vaccination appointments.  
- **💾 Patient Records:** Store and retrieve detailed vaccination history.  
- **📊 Reporting:** Generate simple analytics and reports for vaccine coverage.
