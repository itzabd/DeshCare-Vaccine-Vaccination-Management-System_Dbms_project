## 📋 Project Overview

**DeshCare** is a comprehensive web-based **Vaccination Management System** designed to streamline vaccine distribution, patient registration, and vaccination tracking. This system is built as a Database Management Systems (DBMS) project to efficiently manage healthcare operations for vaccination programs.

The platform enables healthcare providers, administrators, and patients to collaborate seamlessly in managing vaccination schedules, patient records, and vaccination history in a centralized system.

---

## 🎯 Key Objectives

- Digitize vaccination management processes
- Maintain accurate patient vaccination records
- Schedule and track vaccination appointments
- Generate vaccination coverage reports
- Provide role-based access to different user types
- Ensure data integrity and security

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML, CSS, and JavaScript
- Database server (as per project specifications)

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/itzabd/DeshCare-Vaccine-Vaccination-Management-System_Dbms_project.git
   cd DeshCare-Vaccine-Vaccination-Management-System_Dbms_project
   ```

2. **Open in Browser**
   - Navigate to the project directory
   - Open the main HTML file in your preferred web browser
   - Or set up a local web server for better experience

3. **Database Setup**
   - Follow the database schema provided in the project
   - Configure database connection parameters
   - Initialize the database with seed data

## 💾 Database Schema

The system utilizes a relational database with the following main entities:

- **Patients:** Personal and health information
- **Vaccinations:** Vaccine types and details
- **Appointments:** Scheduled vaccination sessions
- **Centers:** Vaccination center information
- **Users:** System users and authentication
- **Records:** Vaccination history and records
